# Setup Guide

## Installing .NET SDK
Download and install the latest .NET SDK from Microsoft. Verify with dotnet --version.

## Installing MySQL
Install MySQL 8 or newer, start the MySQL service, and connect with MySQL Workbench or the MySQL command line.

## Installing VS Code
Install Visual Studio Code, open the Upskill-Program folder, and install the recommended extensions.

## Required Extensions
The .vscode/extensions.json file lists all required extensions for C#, Live Server, formatting, Docker, and Copilot support.

## Running SQL Scripts
Run Module-02-Database/MySql-Exercises/schema.sql, then sample_data.sql, then individual files in solutions/.

## Running C# Projects
Each C# exercise is an independent .NET 8 console app. Open the exercise folder and run dotnet run.

## Running HTML Projects with Live Server
Right-click an index.html file and choose Open with Live Server. Bootstrap and jQuery exercises use CDN links, so internet access is needed for those libraries.
